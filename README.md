# wildlife-tour-app

This is a simple app that allows users to book wildlife tours and view wildlife tours.

# Setup

```
cd Wildlife
```

## Migrations
```
python manage.py makemigrations

python manage.py migrate
```
## Run server
```
python manage.py runserver
```
